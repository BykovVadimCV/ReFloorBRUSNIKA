# ReFloor Brusnika

**Гибридная система автоматического анализа планировок квартир формата "Брусника"**

## Описание

**ReFloor Brusnika** — комплексное решение для автоматической обработки и анализа планов этажей схожего формата от застройщика "Брусника". Система использует гибридный подход, сочетающий классические методы компьютерного зрения с глубоким обучением для точной детекции архитектурных элементов и измерения размеров помещений.

### Основные возможности

- ✅ **Распознавание стен** по цветовой фильтрации с морфологической очисткой
- ✅ **Распознавание дверей и окон** через YOLOv9 (dual-model approach)
- ✅ **Измерение размеров помещений** с автоматическим расчётом масштаба
- ✅ **OCR текстовых меток** (docTR) с удалением аннотаций
- ✅ **Экспорт в Blueprint3D JSON** для 3D-редакторов
- ✅ **Генератор синтетических датасетов** для обучения моделей
- ✅ **Raytracing контуров квартиры** по исходному изображению
- ✅ **Автоматическая классификация** внешних/внутренних стен

## Архитектура системы

### Пайплайн обработки
```
Входное изображение
    ↓
[1] OCR детекция и удаление текста (docTR)
    ↓
[2] Цветовая фильтрация стен (RGB + HSV)
    ↓
[3] YOLO детекция дверей/окон (Model 1 + Model 2)
    ↓
[4] Raytracing контура квартиры
    ↓
[5] Морфологическая обработка и скелетизация
    ↓
[6] Измерение толщины стен (Distance Transform)
    ↓
[7] OCR площади помещения → расчёт масштаба
    ↓
[8] Классификация внешних стен (outer crust)
    ↓
[9] Экспорт JSON + визуализации
```

## Быстрый старт

### Установка
```bash
# Клонирование репозитория
git clone https://github.com/BykovVadimCV/ReFloorBRUSNIKA.git
cd ReFloorBRUSNIKA

# Создание виртуального окружения
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# или
.venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -r requirements.txt

# Установка YOLOv9 (в папку utils/)
cd utils
git clone https://github.com/WongKinYiu/yolov9.git
cd yolov9
pip install -r requirements.txt
cd ../..
```

### Загрузка весов моделей

Скачайте предобученные веса YOLO и поместите в папку `weights/`:

📥 [YANDEX DISK - Модели YOLO](https://disk.yandex.com/d/BTVHG7RsxPjtTw)

### Использование

#### 1️⃣ Основной анализ плана
```bash
python main.py \
  --input input/plan.png \
  --output output/ \
  --yolo-model weights/model2.pt \
  --wall-color 8E8780 \
  --color-tolerance 3 \
  --yolo-conf 0.5 \
  --device cuda  # или cpu
```

**Параметры:**
- `--input` — путь к изображению или папке
- `--output` — директория для результатов
- `--yolo-model` — путь к весам YOLOv9
- `--wall-color` — HEX цвет стен (по умолчанию 8E8780)
- `--auto-wall-color` — автоматическое определение цвета стен
- `--skip-color` — пропустить цветовую детекцию (только YOLO)
- `--disable-measurements` — отключить измерения размеров
- `--no-vis` — не сохранять визуализации
- `--device` — устройство (cpu/cuda/0/1/...)

#### 2️⃣ Генерация синтетического датасета
```bash
python datasetgen.py \
  --output synth_dataset/ \
  --count 1000 \
  --size 1024 \
  --seed 42 \
  --min-wall-thickness 12 \
  --max-wall-thickness 30
```

**Особенности генератора:**
- Реалистичные планировки (кухня, ванная, коридор, жилые комнаты)
- Разнообразные окна: стандартные, широкие, узкие, балконные двери (1-4 секции)
- Двери с дугами открывания различных типов
- Мебель и радиаторы под окнами
- Аугментации: поворот, blur, шум, JPEG-артефакты
- YOLO формат (классы: 0=door, 1=window)

## Технический стек

### Глубокое обучение
- **YOLOv9** (gelan-c) — детекция дверей и окон
- **docTR** (db_resnet50 + crnn_vgg16_bn) — OCR текстовых меток

### Компьютерное зрение
- **OpenCV** — морфологические операции, Hough Transform, Distance Transform
- **scikit-image** — прецизионная скелетизация
- **NumPy/SciPy** — векторные операции, интерполяция

### Подход к детекции

| Компонент | Метод | Преимущества |
|-----------|-------|-------------|
| **Стены** | Цветовая фильтрация (RGB+HSV) | Точность на чистых планах, независимость от обучения |
| **Двери/Окна** | YOLOv9 (dual-model) | Робастность к вариациям, высокий recall |
| **Контур квартиры** | Raytracing на исходном изображении | Пиксель-точная форма, устойчивость к шуму |
| **Масштаб** | OCR + геометрия | Автоматический расчёт м²/пиксель |

### Визуализации

- `*_summary.png` — компактная сводная фигура (2×3 grid)

### Текущие ограничения
